Documentation for the "Examples of Probability Distributions" ShinyApp

Application Overview

This application provides an interactive basic introduction to eight probability distributions. The application allows the user to customize the plot of each distribution by choosing the mean, standard deviation, range or other related parameters and observing the change of the graphic. Also the user can choose the color and the width of the line. Finally for every probability distribution there is the appropriate wikipedia's link that provides the theory concerning the corresponding probability distribution.

Usage

  1. Modify the mean, standard deviation, range, other parameters, color, line's width
  2. Observe how the displayed plot changes
